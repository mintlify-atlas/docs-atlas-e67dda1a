## Overview

Display Mode is PromoMaster Pro's full-screen promotional display engine. It transforms your playlists into stunning, professionally animated digital signage that cycles through products and announcements with cinematic effects, dynamic theming, and smooth transitions.

<Info>
Display Mode is optimized for large screens, TVs, and digital signage displays. It runs in fullscreen mode with no UI chrome or distractions.
</Info>

## Entering Display Mode

Launch Display Mode from the Dashboard:

1. Build a playlist in **Panel de Control**
2. Select your target store
3. Set slide duration
4. Click **Iniciar Transmisión**

Or use **Quick Launch** to display a single item immediately.

## Auto-Rotation System

Display Mode automatically cycles through playlist items:

```tsx AdDisplay.tsx:75-92
useEffect(() => {
  if (items.length <= 1) return;

  const interval = 100;
  const totalSteps = (duration * 1000) / interval;
  
  const timer = setInterval(() => {
    setProgress((prev) => {
      if (prev >= 100) {
        nextSlide();
        return 0;
      }
      return prev + (100 / totalSteps);
    });
  }, interval);

  return () => clearInterval(timer);
}, [duration, items.length, nextSlide]);
```

Transitions include fade and scale effects:

```tsx AdDisplay.tsx:61-68
const nextSlide = useCallback(() => {
  setShowContent(false);
  setTimeout(() => {
    setCurrentIndex((prev) => (prev + 1) % items.length);
    setProgress(0);
    setShowContent(true);
  }, 500);
}, [items.length]);
```

<Accordion title="Rotation Behavior">
- Playlists with 1 item: No rotation, static display
- Playlists with 2+ items: Automatic rotation based on duration setting
- Smooth 500ms transition between slides
- Infinite loop (returns to first item after last)
</Accordion>

## Best Practices

<AccordionGroup>
  <Accordion title="Display Hardware">
    Use displays with:
    - Minimum 1920x1080 resolution (Full HD)
    - 16:9 aspect ratio
    - High brightness (300+ nits for retail)
    - Good viewing angles (IPS panels recommended)
  </Accordion>
  
  <Accordion title="Content Visibility">
    Test your display from typical viewing distances:
    - Counter displays: 3-6 feet
    - Window displays: 10-20 feet
    - Overhead displays: 15-30 feet
    
    Adjust font sizes and image quality accordingly.
  </Accordion>
  
  <Accordion title="Duration Settings">
    Match duration to viewer behavior:
    - High traffic: 3-5 seconds (faster rotation)
    - Moderate traffic: 5-8 seconds (balanced)
    - Waiting areas: 8-15 seconds (detailed viewing)
  </Accordion>
  
  <Accordion title="Browser Setup">
    For permanent installations:
    - Use Chrome or Edge in kiosk mode
    - Disable power-saving/sleep modes
    - Set browser to auto-load PromoMaster on startup
    - Clear cache periodically
  </Accordion>
  
  <Accordion title="Content Strategy">
    - Update playlists regularly (daily/weekly)
    - Use high-quality, high-resolution images
    - Ensure color contrast for readability
    - Test all content before deploying to public displays
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Display Won't Launch">
    - Ensure playlist has at least one item
    - Verify a store is selected
    - Check that products/announcements still exist (not deleted)
    - Try Quick Launch on a single item to isolate the issue
  </Accordion>
  
  <Accordion title="Images Not Showing">
    - Verify image URLs are accessible
    - Check uploaded images didn't exceed browser storage limits
    - Try re-uploading or using a different image source method
  </Accordion>
  
  <Accordion title="Slow Transitions">
    - Reduce image file sizes
    - Clear browser cache
    - Use URL-based images instead of uploads
    - Close other browser tabs
  </Accordion>
  
  <Accordion title="Ticker Not Scrolling">
    - Verify CSS animations are enabled
    - Check for browser compatibility (modern browsers only)
    - Refresh the page
  </Accordion>
</AccordionGroup>
