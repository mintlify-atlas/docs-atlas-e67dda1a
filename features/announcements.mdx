## Overview

Announcements (Avisos Libres) are custom text-based slides that display important messages, store hours, special events, or any information you want to communicate. Unlike product advertisements, announcements give you complete control over typography, colors, alignment, and animation.

<Info>
Announcements can be mixed with product slides in playlists to create a balanced promotional experience.
</Info>

## Accessing Announcements

Navigate to **Gestión Total** (Management) from the sidebar, then select the **Avisos Libres** tab.

## Creating a New Announcement

<Steps>
  <Step title="Enter Internal Title">
    The title is for internal organization only and doesn't appear on the display. Use descriptive names like "Summer Hours" or "Holiday Closure Notice".
  </Step>
  
  <Step title="Write Your Message">
    Enter the text that will appear on screen. This is the main content of your announcement.
  </Step>
  
  <Step title="Choose Typography">
    Select from four font families:
    - **Impactante (Bebas)**: Bold, attention-grabbing display font
    - **Limpio (Inter)**: Clean, modern sans-serif
    - **Elegante (Serif)**: Traditional, formal serif font
    - **Técnico (Mono)**: Monospace font for technical content
  </Step>
  
  <Step title="Adjust Font Size">
    Use the slider to set font size from 20px to 200px. Preview updates in real-time.
  </Step>
  
  <Step title="Set Text Alignment">
    Choose left, center, or right alignment using the alignment buttons.
  </Step>
  
  <Step title="Select Colors">
    - **Fondo** (Background): The slide background color
    - **Letra** (Text): The text color
  </Step>
  
  <Step title="Save Announcement">
    Click **Guardar Aviso** to save your announcement.
  </Step>
</Steps>

## Announcement Data Structure

```typescript types.ts
export interface Announcement {
  id: string;
  title: string;
  message: string;
  fontSize: number; // in pixels
  fontFamily: FontType;
  backgroundColor: string;
  textColor: string;
  textAlign: 'left' | 'center' | 'right';
  hasAnimation: boolean;
}
```

## Text Alignment

Three alignment options are available:

```tsx Management.tsx:323-333
<div className="col-span-1">
  <label className="text-xs font-black text-slate-400 uppercase tracking-widest">Alineación</label>
  <div className="flex gap-1 mt-2">
    {(['left', 'center', 'right'] as const).map(align => (
      <button key={align} type="button" 
              onClick={() => setAnnForm({...annForm, textAlign: align})} 
              className={`flex-1 p-2 rounded-lg border transition-all 
                         ${annForm.textAlign === align ? 'bg-indigo-600 text-white border-indigo-600' : 'bg-white text-slate-400 border-slate-200'}`}>
        {align === 'left' ? <AlignLeft size={16} /> : align === 'center' ? <AlignCenter size={16} /> : <AlignRight size={16} />}
      </button>
    ))}
  </div>
</div>
```

- **Left**: Professional, easy to read for longer content
- **Center**: Dramatic, attention-grabbing for headlines
- **Right**: Unique, artistic for creative messaging

## Best Practices

<AccordionGroup>
  <Accordion title="Use Announcements Strategically">
    Mix 1-2 announcements with 5-10 product slides for optimal balance. Too many announcements can reduce product exposure.
  </Accordion>
  
  <Accordion title="Keep Messages Brief">
    Digital signage viewers have limited attention spans. Aim for messages that can be read in 3-5 seconds.
  </Accordion>
  
  <Accordion title="High Contrast Colors">
    Use strong contrast ratios (dark text on light backgrounds or vice versa). Avoid mid-tone combinations.
  </Accordion>
  
  <Accordion title="Test Readability">
    View your announcements in Display Mode from 10+ feet away. If you can't read it from across the room, increase font size.
  </Accordion>
  
  <Accordion title="Update Regularly">
    Keep announcements current. Remove outdated event announcements and seasonal messages promptly.
  </Accordion>
</AccordionGroup>
