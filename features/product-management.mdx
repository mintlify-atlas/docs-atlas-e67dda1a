## Overview

Product Management is the core feature for creating and organizing your product catalog. Each product can include images, pricing, promotional flags, AI-generated slogans, and custom theming for a stunning display experience.

<Info>
Products created here become available in the Dashboard for building playlists and launching promotional displays.
</Info>

## Accessing Product Management

Navigate to **Gestión Total** (Management) from the sidebar. The **Productos** (Products) tab is selected by default.

## Creating a New Product

<Steps>
  <Step title="Enter Product Name">
    The product name is the only required field. This will be the headline on your promotional display.
  </Step>
  
  <Step title="Add Product Image">
    Choose one of three image input methods:
    - **URL**: Paste an image link
    - **Subir** (Upload): Select a local file from your device
    - **AI**: Generate an image using artificial intelligence
  </Step>
  
  <Step title="Set Price and Unit">
    Enter the price and select the appropriate unit type:
    - Unidad (Unit)
    - Kilo (kg)
    - Gramos (g)
    - Litro (Liter)
    - Pack
  </Step>
  
  <Step title="Add a Slogan (Optional)">
    Enter a custom slogan or click the sparkle icon to generate one with AI.
  </Step>
  
  <Step title="Configure Promotion Settings">
    Check **ES UNA OFERTA** to mark as a promotion and select the type:
    - Oferta Simple (General Promotion)
    - Oferta del Día (Deal of the Day)
    - Oferta Semanal (Weekly Deal)
  </Step>
  
  <Step title="Save Product">
    Click **Crear Producto** to add it to your catalog.
  </Step>
</Steps>

## Image Input Methods

Product images significantly enhance promotional displays. PromoMaster Pro offers three flexible input methods:

### URL Method

<Accordion title="Using Image URLs">
Paste any publicly accessible image URL. Perfect for using product photos from your existing website or CDN.

```tsx Management.tsx:164-166

```
</Accordion>

### Upload Method

<Accordion title="Uploading Local Files">
Upload images directly from your device. Files are converted to Base64 format and embedded in the app data.

```tsx Management.tsx:77-86
const handleFileUpload = (e: React.ChangeEvent<HTMLInputElement>) => {
  const file = e.target.files?.[0];
  if (file) {
    const reader = new FileReader();
    reader.onloadend = () => {
      setProductForm(prev => ({ ...prev, imageUrl: reader.result as string }));
    };
    reader.readAsDataURL(file);
  }
};
```

<Warning>
Uploaded images are stored as Base64 strings in localStorage. Large images may impact performance.
</Warning>
</Accordion>

### AI Generation

<Accordion title="Generating Images with AI">
Click **Generar con IA** to create product images using Google's Gemini AI model. The AI generates images based on the product name.

```tsx Management.tsx:88-97
const handleGenerateImage = async () => {
  if (!productForm.name) {
    alert("Introduce un nombre de producto para generar una imagen.");
    return;
  }
  setIsGeneratingImage(true);
  const imageUrl = await generateProductImage(productForm.name);
  if (imageUrl) setProductForm(prev => ({ ...prev, imageUrl }));
  setIsGeneratingImage(false);
};
```

<Note>
AI image generation requires a valid Gemini API key configured in the application.
</Note>
</Accordion>

## Editing Products

Click the edit icon on any product card to:

1. Populate the form with existing product data
2. Modify any field as needed
3. Click **Actualizar Producto** to save changes

The form button changes from "Crear Producto" to "Actualizar Producto" in edit mode.

## Theme Colors

Each product has a `primaryColor` property (defaults to `#4f46e5` - indigo) that controls:

- Background glow effects in Display Mode
- Badge and accent colors
- Interactive element highlights

<Note>
Custom theme color selection is available in the product form. The color influences the entire aesthetic of the product's promotional display.
</Note>

## Best Practices

<AccordionGroup>
  <Accordion title="Use High-Quality Images">
    Product images are displayed at large sizes in Display Mode. Use images with at least 800x800px resolution for best results.
  </Accordion>
  
  <Accordion title="Keep Names Concise">
    Product names appear in very large typography. Names with 1-3 words work best: "Bife Angus Premium" rather than "Corte de Bife de Lomo de Angus Importado Premium".
  </Accordion>
  
  <Accordion title="Write Compelling Slogans">
    Slogans add emotional appeal. Use action words and benefits: "¡Frescura que se nota!" or "El sabor que mereces".
  </Accordion>
  
  <Accordion title="Strategic Promotion Types">
    Use "Deal of the Day" for urgency, "Weekly Deal" for extended campaigns, and "General Promotion" for evergreen offers.
  </Accordion>
</AccordionGroup>
