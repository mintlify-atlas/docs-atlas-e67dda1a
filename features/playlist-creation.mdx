## Overview

Playlist Creation is the heart of PromoMaster Pro's Dashboard. Build custom sequences by mixing products and announcements, control the order they appear, set duration timing, and launch synchronized displays across your store network.

<Info>
Playlists are temporary and session-based. They're designed for real-time content curation rather than saved templates.
</Info>

## Accessing the Playlist Builder

Navigate to **Panel de Control** (Dashboard) from the sidebar. The playlist builder is accessible via the **Mi Secuencia** (My Sequence) tab.

## Building a Playlist

<Steps>
  <Step title="Browse Content">
    Switch between the **Productos** and **Avisos Libres** tabs to see available items.
  </Step>
  
  <Step title="Add Items">
    Click the **+** button on any product or announcement card to add it to your playlist.
  </Step>
  
  <Step title="Review Sequence">
    Navigate to the **Mi Secuencia** tab to see all added items in order.
  </Step>
  
  <Step title="Reorder Items">
    Use the up/down arrow buttons to adjust the display sequence.
  </Step>
  
  <Step title="Configure Settings">
    - Select your target store from the dropdown
    - Set the duration (in seconds) for each slide
  </Step>
  
  <Step title="Launch Display">
    Click **Iniciar Transmisión** to start the full-screen promotional display.
  </Step>
</Steps>

## Reordering Items

Move items up or down in the sequence:

```tsx Dashboard.tsx:39-45
const movePlaylistItem = (index: number, direction: 'up' | 'down') => {
  const newPlaylist = [...playlist];
  const targetIndex = direction === 'up' ? index - 1 : index + 1;
  if (targetIndex < 0 || targetIndex >= newPlaylist.length) return;
  [newPlaylist[index], newPlaylist[targetIndex]] = [newPlaylist[targetIndex], newPlaylist[index]];
  setPlaylist(newPlaylist);
};
```

<Accordion title="Reordering Strategies">
- **Front-load promotions**: Put time-sensitive deals first
- **Alternate content types**: Mix products and announcements for variety
- **Strategic pacing**: Place high-value items after attention-grabbing content
- **End strong**: Put compelling offers at the end to encourage action
</Accordion>

## Removing Items

Remove individual items without affecting the rest of the sequence:

```tsx Dashboard.tsx:35-37
const removeFromPlaylist = (index: number) => {
  setPlaylist(prev => prev.filter((_, i) => i !== index));
};
```

Click the **X** button on any playlist item card.

  />

<Note>
Duration applies uniformly to all items in the playlist. If you need different durations, launch separate display sessions.
</Note>

### Recommended Duration Settings

- **3-5 seconds**: Fast-paced, high-energy for busy retail environments
- **5-8 seconds**: Balanced timing for mixed content (default: 5)
- **8-12 seconds**: Slower pace for detailed products or complex announcements
- **15+ seconds**: Very long-form content or waiting room displays

## Data Flow to Display

When you launch a display, the playlist data flows to Display Mode:

```tsx Dashboard.tsx:10-11
interface DashboardProps {
  onStartDisplay: (items: DisplayItem[], storeId: string, duration: number) => void;
}
```

The `DisplayItem` type references product/announcement IDs:

```typescript types.ts:39-42
export interface DisplayItem {
  type: 'product' | 'announcement';
  id: string;
}
```

App.tsx resolves these IDs to full data objects:

```tsx App.tsx:112-116
const resolvedItems = (state.currentItems || []).map(item => {
  if (item.type === 'product') return state.products.find(p => p.id === item.id);
  return state.announcements.find(a => a.id === item.id);
}).filter(Boolean) as (Product | Announcement)[];
```

## Best Practices

<AccordionGroup>
  <Accordion title="Optimal Playlist Length">
    Create playlists with 5-15 items. Shorter playlists keep content fresh through repetition; longer playlists provide more variety but may bore viewers before cycling.
  </Accordion>
  
  <Accordion title="Mix Content Types">
    Alternate between products and announcements. A good ratio is 5-7 products for every 1-2 announcements.
  </Accordion>
  
  <Accordion title="Strategic Sequencing">
    - Start with an attention-grabbing promotion or announcement
    - Place high-priority items in positions 2-3 (after initial attention)
    - End with a strong call-to-action or store information
  </Accordion>
  
  <Accordion title="Regular Updates">
    Refresh your playlist daily or weekly. Stale content reduces effectiveness of digital signage.
  </Accordion>
  
  <Accordion title="Test Before Launching">
    Use Quick Launch to preview individual items. Check for typos, image quality, and timing before launching full playlists.
  </Accordion>
</AccordionGroup>
