## Overview

The `Dashboard` component provides the main interface for browsing products and announcements, building custom playlists, and launching ad displays. It features tabbed navigation, search functionality, and drag-to-reorder playlist management.

## Props Interface

```tsx
interface DashboardProps {
  products: Product[];
  announcements: Announcement[];
  stores: Store[];
  onStartDisplay: (items: DisplayItem[], storeId: string, duration: number) => void;
  defaultDuration: number;
}
```

### Props

<ParamField path="products" type="Product[]" required>
  Array of available products to display
</ParamField>

<ParamField path="announcements" type="Announcement[]" required>
  Array of custom announcements/messages
</ParamField>

<ParamField path="stores" type="Store[]" required>
  Array of store locations for display selection
</ParamField>

<ParamField path="onStartDisplay" type="function" required>
  Callback to initiate fullscreen display mode
  
  **Signature**: `(items: DisplayItem[], storeId: string, duration: number) => void`
  
  - `items`: Array of DisplayItems to show in sequence
  - `storeId`: ID of the store to display branding for
  - `duration`: Seconds per slide
</ParamField>

<ParamField path="defaultDuration" type="number" required>
  Default duration in seconds for each slide (typically 5)
</ParamField>

## Key Methods

### Playlist Management

#### addToPlaylist

```tsx
const addToPlaylist = (item: DisplayItem) => {
  setPlaylist(prev => [...prev, item]);
};
```

Adds a single item to the end of the playlist.

#### removeFromPlaylist

```tsx
const removeFromPlaylist = (index: number) => {
  setPlaylist(prev => prev.filter((_, i) => i !== index));
};
```

Removes an item at a specific index from the playlist.

#### movePlaylistItem

```tsx
const movePlaylistItem = (index: number, direction: 'up' | 'down') => {
  const newPlaylist = [...playlist];
  const targetIndex = direction === 'up' ? index - 1 : index + 1;
  if (targetIndex < 0 || targetIndex >= newPlaylist.length) return;
  [newPlaylist[index], newPlaylist[targetIndex]] = 
    [newPlaylist[targetIndex], newPlaylist[index]];
  setPlaylist(newPlaylist);
};
```

Reorders items in the playlist by swapping positions.

#### addAllToPlaylist

```tsx
const addAllToPlaylist = () => {
  const allProducts: DisplayItem[] = products.map(p => 
    ({ type: 'product', id: p.id })
  );
  const allAnns: DisplayItem[] = announcements.map(a => 
    ({ type: 'announcement', id: a.id })
  );
  setPlaylist([...allProducts, ...allAnns]);
  setActiveTab('playlist');
};
```

Adds all products and announcements to the playlist and switches to playlist view.

### Display Control

#### handleLaunchAllMixed

```tsx
const handleLaunchAllMixed = () => {
  if (playlist.length > 0 && selectedStore) {
    onStartDisplay(playlist, selectedStore, duration);
  }
};
```

Launches the full playlist in the selected store.

#### handleLaunchItemNow

```tsx
const handleLaunchItemNow = (item: DisplayItem) => {
  onStartDisplay([item], selectedStore, duration);
};
```

Immediately launches a single item for display.

### Search Filtering

#### filteredProducts

```tsx
const filteredProducts = products.filter(p => 
  p.name.toLowerCase().includes(searchTerm.toLowerCase()) || 
  p.category.toLowerCase().includes(searchTerm.toLowerCase())
);
```

Filters products by name or category matching the search term.

#### filteredAnnouncements

```tsx
const filteredAnnouncements = announcements.filter(a =>
  a.title.toLowerCase().includes(searchTerm.toLowerCase()) ||
  a.message.toLowerCase().includes(searchTerm.toLowerCase())
);
```

Filters announcements by title or message content.

## UI Sections

### Header Controls

The dashboard header includes:

- **Duration Input**: Numeric input for seconds per slide (min: 1)
- **Store Selector**: Dropdown to choose the target store
- **Title and Description**: "Panel de Control" with subtitle

### Navigation Tabs

Three main tabs:

1. **Productos**: Browse and add products
2. **Avisos Libres**: Browse and add announcements  
3. **Mi Secuencia**: View and manage the playlist (shows item count badge)

### Products Tab

**Features**:
- Grid layout (responsive: 1-4 columns based on screen size)
- Product cards with image, name, category, and price
- Hover actions:
  - Play button: Launch item immediately
  - Plus button: Add to playlist
- Search bar for filtering
- "Add All to Sequence" button

**Product Card Data**:
```tsx
<div className="group bg-white rounded-[2.5rem] ...">
  <img src={product.imageUrl || `https://picsum.photos/seed/${product.id}/400`} />
  <h4>{product.name}</h4>
  <p>{product.category}</p>
  <div className="text-2xl">$ {product.price.toLocaleString()}</div>
  {/* Action buttons */}
</div>
```

### Announcements Tab

**Features**:
- Grid layout (1-3 columns)
- Announcement cards with colored icon, title, message preview
- Font family and size badges
- Hover actions:
  - Play button: Launch immediately
  - Plus button: Add to playlist

### Playlist Tab

**Features**:
- Linear list view with index numbers
- Item preview with thumbnail/icon
- Reorder controls (up/down arrows)
- Remove button
- "Vaciar" (Clear) button
- "Iniciar Transmisión" (Start Broadcast) button
- Empty state with "Add All Catalog" quick action

**Playlist Item Rendering**:


## Usage Example

```tsx
import Dashboard from './components/Dashboard';
import { useState } from 'react';

function App() {
  const [products, setProducts] = useState([]);
  const [announcements, setAnnouncements] = useState([]);
  const [stores, setStores] = useState([
    { id: '1', name: 'Main Store', address: '123 Main St', logoColor: 'bg-blue-600' }
  ]);

  const handleStartDisplay = (items, storeId, duration) => {
    // Navigate to display view
    console.log('Starting display:', { items, storeId, duration });
  };

  return (
    <Dashboard
      products={products}
      announcements={announcements}
      stores={stores}
      onStartDisplay={handleStartDisplay}
      defaultDuration={5}
    />
  );
}
```
