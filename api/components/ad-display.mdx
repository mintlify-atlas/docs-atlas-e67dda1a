## Overview

The `AdDisplay` component renders a fullscreen, immersive advertisement display with automatic carousel rotation, animated transitions, and dynamic theming. It supports both product promotions and custom announcements with a professional TV-style presentation.

## Props Interface

```tsx
interface AdDisplayProps {
  items: (Product | Announcement)[];
  store: Store;
  duration: number;
  onExit: () => void;
}
```

### Props

<ParamField path="items" type="(Product | Announcement)[]" required>
  Array of items to display in the carousel. Must contain at least one item.
</ParamField>

<ParamField path="store" type="Store" required>
  Store information for branding (name, address, logo color)
</ParamField>

<ParamField path="duration" type="number" required>
  Duration in seconds for each slide before auto-advancing
</ParamField>

<ParamField path="onExit" type="() => void" required>
  Callback function to exit fullscreen display mode
</ParamField>

## Component State

<ParamField path="currentIndex" type="number">
  Index of the currently displayed item (default: 0)
</ParamField>

<ParamField path="showContent" type="boolean">
  Controls fade-in animation of content (default: false)
</ParamField>

<ParamField path="progress" type="number">
  Progress percentage (0-100) for the progress bar
</ParamField>

## Computed Values

### Current Item Detection

```tsx
const currentItem = items[currentIndex];
const isProduct = 'price' in currentItem;
const product = isProduct ? (currentItem as Product) : null;
const announcement = !isProduct ? (currentItem as Announcement) : null;
```

The component uses TypeScript type guards to determine whether the current item is a Product or Announcement.

### Theme Color

```tsx
const themeColor = product 
  ? (product.primaryColor || '#4f46e5') 
  : (announcement?.backgroundColor || '#1e293b');
```

Dynamically extracts theme color from product or announcement for consistent styling.

### Promotion Labels

```tsx
const getPromoLabel = (type?: string) => {
  switch (type) {
    case 'dia': return '¡OFERTA DEL DÍA!';
    case 'semana': return '¡OFERTA DE LA SEMANA!';
    case 'general':
    default: return '¡OFERTA IMPERDIBLE!';
  }
};
```

## Key Methods

### nextSlide

```tsx
const nextSlide = useCallback(() => {
  setShowContent(false);
  setTimeout(() => {
    setCurrentIndex((prev) => (prev + 1) % items.length);
    setProgress(0);
    setShowContent(true);
  }, 500);
}, [items.length]);
```

Handles transition to the next slide with fade-out/fade-in animation.

**Flow**:
1. Fade out current content (`setShowContent(false)`)
2. Wait 500ms for animation
3. Update index (wraps to 0 after last item)
4. Reset progress bar
5. Fade in new content

## Effects

### Initial Content Display

```tsx
useEffect(() => {
  const timer = setTimeout(() => setShowContent(true), 300);
  return () => clearTimeout(timer);
}, []);
```

Fades in content 300ms after mount.

### Auto-Advance Timer

```tsx
useEffect(() => {
  if (items.length <= 1) return;

  const interval = 100;
  const totalSteps = (duration * 1000) / interval;
  
  const timer = setInterval(() => {
    setProgress((prev) => {
      if (prev >= 100) {
        nextSlide();
        return 0;
      }
      return prev + (100 / totalSteps);
    });
  }, interval);

  return () => clearInterval(timer);
}, [duration, items.length, nextSlide]);
```

**Behavior**:
- Only runs if multiple items exist
- Updates progress bar every 100ms
- Calculates increments based on duration
- Triggers `nextSlide()` when reaching 100%
- Cleans up timer on unmount

## Ticker Messages

Dynamic ticker tape messages generated with `useMemo`:

```tsx
const tickerMessages = useMemo(() => {
  if (isProduct && product) {
    const messages = [
      product.name.toUpperCase(),
      "CALIDAD GARANTIZADA",
      "DISPONIBLE AHORA",
      store.name.toUpperCase()
    ];
    if (product.isPromotion) {
      messages.push("¡OFERTA LIMITADA!");
      messages.push(`MEJOR PRECIO: $${product.price.toLocaleString()}`);
    }
    if (product.category?.toLowerCase().includes('carne')) {
      messages.push("CALIDAD PREMIUM SELECCIONADA");
    }
    return messages;
  } else if (announcement) {
    return [
      announcement.title.toUpperCase(),
      "AVISO IMPORTANTE",
      store.name.toUpperCase(),
      "¡GRACIAS POR SU VISITA!"
    ];
  }
  return [];
}, [currentItem, store.name, isProduct, product, announcement]);
```

**Product Messages**:
- Product name
- "CALIDAD GARANTIZADA"
- "DISPONIBLE AHORA"
- Store name
- Promotion-specific messages (if applicable)
- Category-specific messages (e.g., meat quality)

**Announcement Messages**:
- Announcement title
- "AVISO IMPORTANTE"
- Store name
- "¡GRACIAS POR SU VISITA!"

## Slide Indicators

```tsx
{items.length > 1 && (
  <div className="absolute bottom-20 left-1/2 -translate-x-1/2 flex gap-3">
    {items.map((_, i) => (
      <div 
        key={i} 
        className={`h-2 rounded-full cursor-pointer ${
          i === currentIndex ? 'w-12 bg-white' : 'w-2 bg-white/10'
        }`}
        style={{ backgroundColor: i === currentIndex ? themeColor : undefined }}
        onClick={() => {
          setCurrentIndex(i);
          setProgress(0);
          setShowContent(false);
          setTimeout(() => setShowContent(true), 100);
        }}
      />
    ))}
  </div>
)}
```

**Features**:
- Clickable navigation
- Current indicator is elongated (48px vs 8px)
- Theme-colored when active
- Only shown for multiple items

## Usage Example

```tsx
import AdDisplay from './components/AdDisplay';
import { useState } from 'react';

function App() {
  const [isDisplaying, setIsDisplaying] = useState(false);
  
  const products = [
    {
      id: '1',
      name: 'Carne Premium',
      price: 2500,
      unit: 'kg',
      isPromotion: true,
      promotionType: 'dia',
      slogan: 'La mejor calidad para tu familia',
      imageUrl: '/images/carne.jpg',
      primaryColor: '#dc2626',
      category: 'Carnes',
      description: ''
    }
  ];

  const store = {
    id: '1',
    name: 'Supermercado Central',
    address: 'Av. Libertador 1234',
    logoColor: 'bg-blue-600'
  };

  if (isDisplaying) {
    return (
      <AdDisplay
        items={products}
        store={store}
        duration={5}
        onExit={() => setIsDisplaying(false)}
      />
    );
  }

  return (
    <button onClick={() => setIsDisplaying(true)}>
      Start Display
    </button>
  );
}
```

- **Tablet**: Medium image (448px), adjusted spacing
- **Desktop**: Large image (512px), horizontal layout with wide spacing
