## Overview

The `Management` component provides a comprehensive admin interface for managing stores, products, and announcements. It features AI-powered slogan generation, image generation via Gemini API, and real-time announcement preview.

## Props Interface

```tsx
interface ManagementProps {
  stores: Store[];
  products: Product[];
  announcements: Announcement[];
  onAddStore: (store: Omit<Store, 'id'>) => void;

  onDeleteStore: (id: string) => void;

  onDeleteProduct: (id: string) => void;
```

## Key Methods

### Product Management

#### handleProductSubmit

```tsx
const handleProductSubmit = async (e: React.FormEvent) => {
  e.preventDefault();
  if (editingProductId) {
    onUpdateProduct({ ...productForm, id: editingProductId });
    setEditingProductId(null);
  } else {
    onAddProduct(productForm);
  }
  setProductForm(INITIAL_PRODUCT_STATE);
  setImgSourceMode('url');
};
```

Handles form submission for both creating and updating products.

#### handleFileUpload

```tsx
const handleFileUpload = (e: React.ChangeEvent<HTMLInputElement>) => {
  const file = e.target.files?.[0];
  if (file) {
    const reader = new FileReader();
    reader.onloadend = () => {
      setProductForm(prev => ({ ...prev, imageUrl: reader.result as string }));
    };
    reader.readAsDataURL(file);
  }
};
```

Converts uploaded image files to base64 data URLs for storage.

#### handleGenerateImage

```tsx
const handleGenerateImage = async () => {
  if (!productForm.name) {
    alert("Introduce un nombre de producto para generar una imagen.");
    return;
  }
  setIsGeneratingImage(true);
  const imageUrl = await generateProductImage(productForm.name);
  if (imageUrl) setProductForm(prev => ({ ...prev, imageUrl }));
  setIsGeneratingImage(false);
};
```

Generates product images using the Gemini API based on product name.

### Announcement Management

#### handleAnnSubmit

```tsx
const handleAnnSubmit = (e: React.FormEvent) => {
  e.preventDefault();
  if (editingAnnId) {
    onUpdateAnnouncement({ ...annForm, id: editingAnnId });
    setEditingAnnId(null);
  } else {
    onAddAnnouncement(annForm);
  }
  setAnnForm(INITIAL_ANN_STATE);
};
```

#### handleEditAnn

```tsx
const handleEditAnn = (ann: Announcement) => {
  setEditingAnnId(ann.id);
  setAnnForm({ ...ann });
  window.scrollTo({ top: 0, behavior: 'smooth' });
};
```

Loads announcement data into the form and scrolls to editor.

### AI-Powered Features

#### Slogan Generation

```tsx
<button type="button" onClick={async () => {
  if(!productForm.name) return alert("Indica el nombre del producto.");
  setIsGeneratingSlogan(true);
  const s = await generateSlogan(productForm.name, productForm.description);
  setProductForm(prev => ({ ...prev, slogan: s }));
  setIsGeneratingSlogan(false);
}}>
  {isGeneratingSlogan ? <Loader2 className="animate-spin" /> : <Sparkles />}
</button>
```

Generates marketing slogans using AI based on product name and description.

## UI Sections

### Products Tab

**Layout**: 2-column grid (form + list)

**Form Features**:
- Text input for product name
- Price and unit selection
- Image source selector with 3 modes:
  - **URL**: Direct image link input
  - **Upload**: File picker for local images
  - **AI**: Gemini-powered image generation
- Slogan textarea with AI generation button
- Promotion toggle with type selector (general/día/semana)
- Color picker for theme color

**List Features**:
- Product cards with thumbnail, name, price
- Promotion badge overlay
- Edit and delete actions
- Empty state message

### Announcements Tab

**Layout**: 2-column grid (editor + preview/list)

**Editor Form**:
- Title input (internal reference)
- Message textarea (display content)
- Typography selector:
  - Bebas (Impactante)
  - Sans (Limpio)
  - Serif (Elegante)
  - Mono (Técnico)
- Font size slider (20-200px)
- Text alignment buttons (left/center/right)
- Background color picker
- Text color picker

## Usage Example

```tsx
import Management from './components/Management';
import { useState } from 'react';

function App() {
  const [stores, setStores] = useState([]);
  const [products, setProducts] = useState([]);
  const [announcements, setAnnouncements] = useState([]);

  const addStore = (store) => {
    const newStore = { ...store, id: crypto.randomUUID() };
    setStores([...stores, newStore]);
  };

  const updateStore = (updatedStore) => {
    setStores(stores.map(s => 
      s.id === updatedStore.id ? updatedStore : s
    ));
  };

  const deleteStore = (id) => {
    setStores(stores.filter(s => s.id !== id));
  };

  // Similar handlers for products and announcements...

  return (
    <Management
      stores={stores}
      products={products}
      announcements={announcements}
      onAddStore={addStore}
      onUpdateStore={updateStore}
      onDeleteStore={deleteStore}
      onAddProduct={addProduct}
      onUpdateProduct={updateProduct}
      onDeleteProduct={deleteProduct}
      onAddAnnouncement={addAnnouncement}
      onUpdateAnnouncement={updateAnnouncement}
      onDeleteAnnouncement={deleteAnnouncement}
    />
  );
}
```
